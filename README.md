# WiFi smart switch

An IoT smart-switch project that allow remote activation of 2 relay channels through different interface solutions. It provides a WiFi connection acessible through: Telegram bot commands, HTML plus Message Queuing Telemetry Transport (MQTT) Broker or local webserver.
